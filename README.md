# AI Resume Analyzer

AI Resume Analyzer is a tool that analyzes a resume PDF and provides insights about skills and resume quality.

## Features
- Upload Resume PDF
- Extract Skills
- Resume Score
- Improvement Suggestions

## Tech Stack
- Python
- Streamlit
- PDFPlumber

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
